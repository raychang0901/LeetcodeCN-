# 哈希表

## 题目链接

https://leetcode-cn.com/problems/best-sightseeing-pair/

力扣1002. 查找常用字符

運用正確的哈希表很直觀~
    
---------------------------------------

```cpp

class Solution {
public:
    vector<string> commonChars(vector<string>& A) {
        vector<vector<int>> hash(A.size(), vector<int>(26, 0));
        vector<string> sol;
        for(int i = 0; i<A.size(); i++){
            for(int j = 0; j<A[i].size(); j++){
                hash[i][A[i][j]-'a']++;
            }
        }
        for(int i = 0; i<26; i++){
            int min = INT_MAX;
            for(int j = 0; j<A.size(); j++){
                if(hash[j][i]<min){
                    min = hash[j][i];
                }
            }
            if(min!=INT_MAX){
                while(min--){
                    string ss(1, 'a'+i);
                    sol.push_back(ss);
                }
            }
        }
        return sol;
    }
};
```
