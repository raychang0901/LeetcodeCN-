# 哈希表

## 题目链接

https://leetcode-cn.com/problems/jewels-and-stones/


##代碼
---------------------------------------

```cpp
class Solution {
public:
    int numJewelsInStones(string J, string S) {
        int hash[126]{0};
        int res = 0;
        for(int i = 0; i<J.size(); i++){
            hash[J[i]]++;
        }
        for(int i = 0; i<S.size(); i++){
            if(hash[S[i]]){
                res++;
            }
        }
        return res;
    }
};
```
