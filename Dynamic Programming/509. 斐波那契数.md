# 動態規劃

## 题目链接

https://leetcode-cn.com/problems/fibonacci-number/submissions/

力扣509. 斐波那契数

動態規劃的第一題，可以了解動態規劃的意義，也最好實做、最直觀~    
    
---------------------------------------

```cpp
class Solution {
public:
    int fib(int n) {
        int fib_arr[31]{0};
        fib_arr[0] = 0;
        fib_arr[1] = 1;
        fib_arr[2] = 1;
        for(int i = 3; i<=n; i++){
            fib_arr[i] = fib_arr[i-1]+fib_arr[i-2];
        }
        return fib_arr[n];
    }
};
```
